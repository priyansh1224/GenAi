ANN Approach;
In the ANN approach, we will use a feedForward neural network to predict the next number in the sequence. The input to the network will be the previous numbers in the sequence, and the output will be the predicted next number.

First Method:
1. Clustering/ Inverted file Index:
In this method, we will use clustering to group similar sequences together. We can then create an inverted file index to quickly retrieve the sequences that are similar to the input sequence. This will allow us to make predictions based on the most similar sequences in the cluster.

2.Binary space Partitioning :
**Binary Space Partitioning (BSP)** is a method used to divide space into smaller regions using lines (2D) or planes (3D).
It works by recursively splitting space into **front and back parts**, forming a tree structure (BSP Tree).
This helps in **efficient rendering**, especially for deciding which objects to draw first.
It was widely used in games like Doom and Quake.
BSP is fast for **static scenes** but not ideal for frequently moving objects.
3. Hirarchical Navigable Small worlds -HNSW

**Hierarchical Navigable Small Worlds (HNSW)** is a graph-based algorithm for efficient approximate nearest neighbor search.

It builds a multi-layer graph where each layer contains a subset of the data points, with higher layers having fewer points.
